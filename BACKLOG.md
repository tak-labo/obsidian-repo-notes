# Backlog

Future feature candidates, not yet scheduled for implementation.

---

## 1. Profile import / export (Size: S–M)

- Export profile settings as JSON, import from JSON
- Useful for migrating to another device or creating backups
- Add Export / Import buttons to the settings tab
- Exclude GitHub token from export (security)

## 2. Note content preview (Size: M)

- Show a sample note generated by the current profile in the settings tab
- Lets users see the effect of setting changes immediately
- Call `buildNote()` with mock data and render the Markdown

## 3. Sync schedule (Size: L)

- Auto-sync at regular intervals (e.g. every X minutes / hours), not just on startup
- Use Obsidian's `registerInterval()` for timer management
- Add an interval input field to the settings tab
