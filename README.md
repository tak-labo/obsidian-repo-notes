# Repo Notes — Obsidian Plugin

> 🇯🇵 [日本語版はこちら](#repo-notes--obsidian-プラグイン)

Import your GitHub **starred repositories**, **own repositories**, and **organization repositories** as structured Obsidian notes, with rich frontmatter properties and optional AI-powered README summaries.

## Features

- ⭐ **GitHub Stars** — sync all starred repos as notes
- 📁 **My Repos** — sync your own public/private repos
- 🏢 **Organizations** — sync repos from specified GitHub organizations
- 👤 **Multi-profile** — manage multiple GitHub accounts as separate profiles
- 🗂 **Rich frontmatter** — URL, description, language, stars, forks, commits, last updated, topics as tags
- 🤖 **AI README summary** — summarize READMEs via Claude (Anthropic API)
- 📄 **Raw README** — embed full README content in notes
- 🌐 **i18n** — UI switches automatically between English and Japanese based on Obsidian's locale

## Installation

### Manual

1. Download the latest release from [Releases](../../releases)
2. Extract `main.js`, `manifest.json`, and `styles.css` into your vault:
   ```
   <vault>/.obsidian/plugins/repo-notes/
   ```
3. In Obsidian → Settings → Community plugins → enable **Repo Notes**

### BRAT (Beta Reviewers Auto-update Tool)

Add this repository URL via the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat).

## Setup

1. Go to **Settings → Repo Notes**
2. Create a profile (e.g. "Personal", "Work")
3. Paste your GitHub Personal Access Token

### Required token scopes

| Feature | Classic PAT | Fine-grained PAT |
|---|---|---|
| Starred repos | `public_repo` | `Starring` (read) |
| Private repos | `repo` | `Repositories` (read) |
| README / commits | `public_repo` | `Contents` (read) |
| Organization repos | `read:org` | Organization → `Members` (read) |

Generate tokens at [github.com/settings/tokens](https://github.com/settings/tokens)

## Note structure

Each repository becomes a single Markdown file with frontmatter properties:

```yaml
---
source: starred          # or "my-repo" / "org-repo"
profile: "Personal"
repo: "owner/repo-name"
url: "https://github.com/owner/repo-name"
website: "https://example.com"
description: "..."
language: TypeScript
stars: 8420
forks: 1203
commits: 3241
last_updated: 2025-03-18
starred_at: 2024-11-15
synced_at: 2025-03-20
tags: ["obsidian", "markdown"]
---
```

Fully queryable via [Dataview](https://github.com/blacksmithgu/obsidian-dataview):

```dataview
TABLE stars, language, last_updated
FROM "GitHub Stars"
WHERE profile = "Personal"
SORT stars DESC
```

## AI README Summary

Enable **Include AI README summary** in settings and add your [Anthropic API key](https://console.anthropic.com).
Summaries are generated by Claude Haiku — fast and cost-effective (a few cents per 100 repos).

## Development

```bash
git clone https://github.com/<you>/obsidian-repo-notes

npm install

```

Symlink the repo into your vault's plugin folder for live development:


# Repo Notes — Obsidian プラグイン

> 🇬🇧 [English version above](#repo-notes--obsidian-plugin)

GitHubの**Starしたリポジトリ**・**自分のリポジトリ**・**Organizationのリポジトリ**を、YAMLフロントマター付きの構造化されたObsidianノートとしてインポートするプラグインです。AIによるREADME要約にも対応しています。

## 機能

- ⭐ **GitHub Stars** — Starしたリポジトリを一括ノート化
- 📁 **自分のRepo** — public/privateリポジトリを同期
- 🏢 **Organization** — 指定したOrganizationのリポジトリを同期
- 👤 **マルチプロファイル** — 複数のGitHubアカウントをプロファイルで管理
- 🗂 **リッチなフロントマター** — URL、説明文、言語、Star数、Fork数、コミット数、最終更新日、トピックをタグとして保存
- 🤖 **AI README要約** — Claude（Anthropic API）でREADMEを自動要約
- 📄 **README全文埋め込み** — READMEの全文をノートに含める
- 🌐 **多言語対応** — ObsidianのロケールにあわせてUIが日英自動切り替え

## インストール

### 手動インストール

1. [Releases](../../releases) から最新版をダウンロード
2. `main.js`・`manifest.json`・`styles.css` をVaultの以下に配置：
   ```
   <vault>/.obsidian/plugins/repo-notes/
   ```
3. Obsidian → 設定 → コミュニティプラグイン → **Repo Notes** を有効化

### BRAT（Beta Reviewers Auto-update Tool）

[BRATプラグイン](https://github.com/TfTHacker/obsidian42-brat) にこのリポジトリのURLを追加することでインストール可能です。

## セットアップ

1. **設定 → Repo Notes** を開く
2. プロファイルを作成（例：「Personal」「Work」）
3. GitHub Personal Access Tokenを貼り付ける

### 必要なトークンスコープ

| 機能 | Classic PAT | Fine-grained PAT |
|---|---|---|
| Star済みリポジトリ | `public_repo` | `Starring`（read） |
| プライベートリポジトリ | `repo` | `Repositories`（read） |
| README / コミット数 | `public_repo` | `Contents`（read） |
| Organizationリポジトリ | `read:org` | Organization → `Members`（read） |

トークンの発行: [github.com/settings/tokens](https://github.com/settings/tokens)

## ノートの構造

リポジトリ1件につき1つのMarkdownファイルが生成されます：

```yaml
---
source: starred          # または "my-repo" / "org-repo"
profile: "Personal"
repo: "owner/repo-name"
url: "https://github.com/owner/repo-name"
website: "https://example.com"
description: "..."
language: TypeScript
stars: 8420
forks: 1203
commits: 3241
last_updated: 2025-03-18
starred_at: 2024-11-15
synced_at: 2025-03-20
tags: ["obsidian", "markdown"]
---
```

[Dataview](https://github.com/blacksmithgu/obsidian-dataview) を使ってノートを検索・集計できます：

```dataview
TABLE stars, language, last_updated
FROM "GitHub Stars"
WHERE profile = "Personal"
SORT stars DESC
```

## AI README要約

設定で **README要約（AI）を含める** をオンにして、[Anthropic APIキー](https://console.anthropic.com) を入力してください。
Claude Haikuで要約を生成します。高速かつ低コスト（100リポジトリあたり数セント程度）。
